CSS Specificity  -> Important > inline styling > id > class > element  (decreasing specificity.)

BEM -> Block , Element , Modifier.

issue/ problems 
1 > specificity issue : avoid id/tag.
2 > classname clashes.
3 > Not a standard name for class.

-> with BEM Architecture we can resolve all above issues.

Block  : we can use letters , digits , dashes for naming block ;
         use dashes when more that one word.
        
Element : we can user letters , digits , dashes , underscore from naming element.
syntax : block__element.

Modifier :  Modifier is something the changes in appearance of state.
            we can user letters , digits , dashes , underscore from naming element.
syntax   : block/element--modifier

state of element.
